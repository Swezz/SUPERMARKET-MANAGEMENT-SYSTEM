# Supermarket Management System

## Description

This project is a console-based supermarket management system developed in C++. It allows users to view available products, add items to a cart, and complete the checkout process. The program is structured using object-oriented programming concepts.

## Features

* Display available products with price and stock
* Add items to cart
* View cart details
* Calculate total bill at checkout

## Technologies Used

* C++
* Object-Oriented Programming
* STL (vector)

## Project Structure

* include/ : header files
* src/ : source files
* main.cpp : entry point of the program

## How to Run

Compile:
g++ main.cpp src/*.cpp -o app

Run:
./app

## Note

Executable files are not included. Compile the program using the command above to run it locally.
